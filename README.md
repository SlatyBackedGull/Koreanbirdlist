# Koreanbirdlist
미디어 아트 기말과제 겸 내가 쓰려고 만드는 사이트

## Supabase 설정

- `app.js` 상단의 `SUPABASE_URL`과 `SUPABASE_KEY` 값을 본인의 Supabase 프로젝트 설정에 맞게 수정하세요.
- `USE_SUPABASE` 플래그를 `true`로 설정해야 Supabase 서비스가 활성화됩니다.

## 로컬 데이터 백업 및 복원

- **데이터 백업**: 메뉴의 '데이터 백업' 버튼을 클릭하면 현재 데이터를 JSON 파일로 다운로드합니다.
- **데이터 복원**: 메뉴의 '데이터 복원' 버튼을 클릭하고 백업 JSON 파일을 선택하면 데이터가 복원되어 Supabase 또는 로컬 저장소에 저장됩니다.

## 사용 방법

1. 사이트에 회원가입 또는 로그인 후 종 목록 추가/수정 화면에서 관찰한 종과 메모, 미디어를 추가합니다.
2. 완료(저장) 버튼을 눌러 Supabase 또는 로컬 저장소에 저장합니다.
3. **백업** 기능으로 로컬에 데이터를 저장해두면, 네트워크 장애 등으로 Supabase 저장 실패 시에도 데이터를 안전하게 보관할 수 있습니다.
4. **복원** 기능으로 백업 파일을 불러와 데이터를 복원하고 다시 Supabase에 업로드할 수 있습니다.
